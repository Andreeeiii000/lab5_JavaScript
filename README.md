## Ответы на вопросы

### 1. Какие семантические теги вы использовали?
Использованы основные семантические теги HTML5:
- `<header>` — шапка сайта с заголовком и меню навигации.  
- `<nav>` — блок с навигационными ссылками.  
- `<main>` — основное содержимое страницы.  
- `<section>` — логический раздел контента.  
- `<article>` — основной текстовый материал.  
- `<aside>` — боковая панель с дополнительной информацией.  
- `<footer>` — подвал сайта с копирайтом.

---

### 2. Где использовали Flexbox, а где Grid?
- **Flexbox** применён в шапке (`header`) для горизонтального размещения заголовка и меню.  
- **Grid** использован в основном блоке (`.content`) для размещения основной статьи (`article`) и боковой панели (`aside`) в две колонки.

---

### 3. Какой медиазапрос добавили и что он меняет?
Добавлен медиазапрос:
```css
@media (max-width: 768px) { ... }

Он делает сайт адаптивным для мобильных устройств:
1) Шапка перестраивается в вертикальное направление.
2) Навигационное меню (.nav-list) отображается в колонку.
3) Контентная область (.content) превращается из двух колонок в одну.
4) Все элементы занимают 100% ширины экрана.
